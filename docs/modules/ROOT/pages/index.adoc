== Installation

If you want to use this extension, you need to add the `io.quarkiverse.pinot:quarkus-pinot` extension first.
In your `pom.xml` file, add:

[source,xml]
----
<dependency>
    <groupId>io.quarkiverse.pinot</groupId>
    <artifactId>quarkus-pinot</artifactId>
</dependency>
----
